About Me

Hi, I’m Josephine — a graduate student at NYU Tandon School of Engineering, pursuing my Master’s in Mechatronics and Robotics Engineering on a fully funded scholarship. Before NYU, I studied in China at Zhejiang University of Science and Technology, where I was honored with the Excellent Foreign Student Scholarship for academic excellence and perseverance. My work blends hands-on robotics with intelligent control systems. I have experience developing robotic platforms using ROS, Python, C++, and embedded systems, and I’m passionate about sensor fusion, autonomy, and assistive technologies. This website is a collection of the projects, research, and ideas that have shaped my journey so far. I’m always excited to connect with fellow engineers, collaborators, and curious minds who share a passion for building the future of robotics. Let’s connect and create something impactful.

Projects

Autonomous Navigation Study using ROS and TurtleBot3

Developed an autonomous navigation system for the TurtleBot3 Waffle Pi using ROS Noetic and C++, integrating SLAM (gmapping), path planning, and obstacle avoidance via the ROS Navigation Stack.

  • ROS
  • Gazebo Simulation
  • Navigation
  • C++

Navigation Systems with RC-Car

Designed and programmed a Bluetooth-controlled mobile robot using Arduino and L298P motor driver, enabling real-time wireless control via Android. Integrated ultrasonic sensors and servo mechanisms for basic indoor obstacle avoidance. Developed modular embedded code in Arduino IDE and explored GPS-based navigation and path planning (A, Dijkstra, RRT) for potential autonomy enhancements.

  • C
  • Obstacle avoidance
  • Navigation

Plate Cleaning Robot

Developed an automated plate-cleaning system with an ABB IRB 120 robotic arm using inverse kinematics and PD control for spiral motions. It features efficient trajectory planning and enhanced reliability through random plate positioning and robust task space design.

  • Matlab
  • ABB IRB 120
  • Inverse Kinematics
  • PD Control
  • Trajectory Planning
  • Automation

Multimodal Calibration Dataset

A versatile multimodal calibration platform that uses robotic arms and a high-precision MoCap system to collect diverse extrinsic data between cameras and LiDAR sensors across both static and dynamic configurations.

  • ROS
  • Camera-LiDAR Calibration
  • Manipulation
  • MoCap

  • Data Collection
